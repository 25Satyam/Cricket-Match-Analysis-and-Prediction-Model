# 🏏 Cricket Match Analysis & Prediction Model

> An interactive ML-powered web application that predicts final scores and win probabilities at any stage of a cricket match using Random Forest models, built with Python and Streamlit.

---

## 📸 Features

| Feature | Description |
|---|---|
| 🎯 **Score Predictor** | Predict the final innings score at any over using live match state |
| 📈 **Win Probability** | Real-time win/loss probability for the chasing team |
| ⚡ **Live Simulator** | Over-by-over win probability graph and runs progression |
| 📊 **Match Comparison** | Head-to-head stats with bar charts and radar visualization |

---

## 🛠️ Tech Stack

- **Python 3.10+**
- **Streamlit** — Interactive web application
- **Pandas & NumPy** — Data manipulation and feature engineering
- **scikit-learn** — Random Forest Regressor & Classifier
- **Plotly** — Interactive charts (gauge, donut, radar, line, bar)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cricket-analysis.git
cd cricket-analysis
```

### 2. Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate        # macOS / Linux
venv\Scripts\activate           # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Generate sample match data

```bash
python generate_data.py
```

This creates `data/t20_matches.csv` and `data/odi_matches.csv` with synthetic match data.

### 5. Run the app

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.

## 🤖 How the Models Work

### Score Predictor (Random Forest Regressor)
Predicts the **final innings score** given the current match state.

**Features used:**
- Overs completed
- Current runs & wickets
- Overs / balls remaining
- Current run rate
- Wickets in hand

**Training:** 2,500 synthetic T20/ODI innings with realistic score distributions and wicket patterns.  
**Metric:** Mean Absolute Error (MAE) ≈ ±8–12 runs

---

### Win Probability Model (Random Forest Classifier)
Predicts **P(chasing team wins)** at any point in a run chase.

**Features used:**
- Target score
- Current runs, wickets, overs
- Runs needed & balls remaining
- Required run rate vs current run rate (RR differential)
- Wickets in hand

**Training:** 2,500 synthetic chases with varied targets and match outcomes.  
**Metric:** Classification Accuracy ≈ 80–85 %
